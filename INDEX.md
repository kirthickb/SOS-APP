## üìä Project Overview at a Glance

### What is this?

A full-stack emergency ambulance application with:

- **React Native mobile app** (CLIENT and DRIVER)
- **Spring Boot backend** with REST API and WebSocket
- **Real-time alerts** for emergency SOS
- **Maps integration** for tracking and navigation
- **JWT authentication**
- **MySQL database**

### Key Features:

- üÜò One-click SOS button for patients
- üöë Real-time driver notifications
- üó∫Ô∏è GPS tracking and navigation
- üì± Mobile app for Android/iOS
- üîê Secure authentication
- ‚ö° WebSocket real-time updates
- üìç OpenStreetMap integration (free)

### Tech Stack:

**Backend**: Spring Boot, MySQL, JWT, WebSocket  
**Frontend**: React Native, Expo, TypeScript, Axios  
**Maps**: react-native-maps, OpenStreetMap, OSRM
